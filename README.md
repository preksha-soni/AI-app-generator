# AI App Generator 🤖

Natural Language → Complete App Schema using Claude AI

## What it does
Converts plain English app descriptions into complete schemas through a 4-stage pipeline:
- Stage 1: Intent Extraction
- Stage 2: System Architecture  
- Stage 3: Schema Generation
- Stage 4: Validation & Repair
